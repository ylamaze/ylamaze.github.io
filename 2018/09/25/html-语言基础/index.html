html 语言基础

发表于 2018-09-25 更新于 2018-09-26 分类于 网页设计 阅读次数： Valine：

HTML是用来编写网页的一种语言，用它编写的文件简称超文本，或者网页文件。HTML是通过利用各种标签来标识文档的结构以及超链接信息的。

标签

标签分单独出现的标签和成对出现的标签两种。大多数标签都是成对出现的，由首标签和尾标签组成。首标签格式：<元素名>，尾标签的格式为</元素名>

HTML语法的一般格式如下：

<标签名 属性名1=属性值1 属性名2=属性值2 …>标签中的具体内容</标签名>

  • < html >标签
    首标签< html>和尾标签< /html>分别位于文档的最前面和最后面，明确地表示文档是以超文本语言（HTML）编写的。

  • < head >标签
    习惯上我们把HTML文档分为文档头和文档主体两个部分。主体部分就是我们在浏览器用户区看到的内容。< head >标签包裹的就是文档头部分。如< title>标签就是嵌套在< head>标签中的子标签，< title>和< /title>之间的内容将显示在Web浏览器窗口的标题栏中。

  • < body>标签
    在< body>< /body>之间的内容将显示在浏览器窗口的用户区内。他是HTML文档夫的主体部分。在< body>
    标签中可以规定整个文档的一些基本属性，如背景颜色，背景图片，字体字号等。

  • < hn>标题标签
    HTML总共提供6个等级的标题，n越小，字号就越大，< h1>定义最大号标题，< h6>定义最小号标题。

  • < br>换行标签
    换行符号标签是单个标签，如果想把一个句子后面内容在下一行显示就会用到换行符< br>。

  • < hr>水平线标签
    用< hr>标签可以在网页上画出一条横跨网页的水平分隔线。

  • < font>字体标签
    < font>字体标签是HTML里最常用文字格式控制标签，通过改变< font>标签的属性可以改变文字的大小，颜色，字体等。< font>标签的主要属性有：

1.size
< size>标签的size属性指定文字的大小，它的取值范围为1~7，当它取值为1时文字最小，7最大，默认值是3.

2.color
< font>标签的color属性指定文字的颜色，它的取值可以用英文关键字，十六进制颜色代码，rgb函数三种类型。

3.face
< font>标签的face属性指定文字的字体。

  • < p>段落标签
    < p>标签符用于划分段落，控制文本位置。< p>是成对标签符，用于定义内容从新的一行开始，并与上段之间有一个空行，其align属性定义新开始的一行内容在页面中的对齐位置，属性值可以是left(左对齐)，center(居中对齐)，或者right（右对齐）。
