极限与连续总结

发表于 2020-03-30 更新于 2020-03-31 分类于 考研 阅读次数： Valine：
汤神的极限与连续部分视频已经看完，总结出如下重要的内容

极限

类型一（N项和求极限）

  • N项相加求极限，分子分母不齐用夹逼定律
  • 齐不动动非齐，齐不齐看次数
    例: \[\begin{array}{l} \mathop {\lim }\limits_{n \to \infty } (\frac{1}{{\sqrt {{n^2} + 1} }} + \frac{1}{{\sqrt {{n^2} + 2} }} + \frac{1}{{\sqrt {{n^2} + 3} }} + ...... + \frac{1}{{\sqrt {{n^2} + n} }})\\ bn = \frac{1}{{\sqrt {{n^2} + 1} }} + \frac{1}{{\sqrt {{n^2} + 2} }} + \frac{1}{{\sqrt {{n^2} + 3} }} + ...... + \frac{1}{{\sqrt {{n^2} + n} }}\\ \frac{n}{{\sqrt {{n^2} + n} }} \le bn \le \frac{n}{{\sqrt {{n^2} + } 1}}\\ \mathop {\lim }\limits_{n \to \infty } \frac{1}{{\sqrt {1 + \frac{1}{n}} }} = 1 = \mathop {\lim }\limits_{n \to \infty } \frac{1}{{\sqrt {1 + \frac{1}{{{n^2}}}} }} = 1\\ \therefore {原式\rm{ = 1}} \end{array}\]
  • 当分子分母都齐，且分母高一次（定积分定义）
    记做:$\mathop {\lim }\limits_{n \to \infty } \frac{1}{n}\sum\limits_{i = 1}^n {f(\frac{1}{n})} = \int\limits_0^1 {f(x)dx}$
    例： \[\begin{array}{l} \mathop {\lim }\limits_{n \to \infty } (\frac{1}{{n + 1}} + \frac{1}{{n + 2}} + ......\frac{1}{{n + n}})\\ \mathop {\lim }\limits_{n \to \infty } \sum\limits_{i = 1}^n {\frac{1}{{n + i}}} \\ = \mathop {\lim }\limits_{n \to \infty } \frac{1}{n}\sum\limits_{i = 1}^n {\frac{1}{{1 + \frac{i}{n}}}} \\ = \int\limits_0^1 {\frac{1}{{1 + x}}} dx\\ = \ln (1 + x)|_0^1\\ = \ln 2 \end{array}\]

类型二 极限存在性证明

  • 单调有界函数必有极限
    例： \[\begin{array}{l} {a_1} = \sqrt 2 ,{a_2} = \sqrt {2 + \sqrt 2 } ,{a_3} = \sqrt {2 + \sqrt {2 + \sqrt 2 } } ......\\ 证：\mathop {\lim }\limits_{n \to \infty } {a_n}\exists，求之\\ 解：{a_{n + 1}} = \sqrt {2 + {a_n}} \\ 现证：{a_n} \le 2\\ {a_{_1}} = \sqrt 2 \le 2\\ 设{a_k} \le 2;{a_{k + 1}} = \sqrt {2 + {a_k}} \le \sqrt {2 + 2} \le 2\\ \therefore {\forall _n},{a_n} \le 2,\mathop {\lim }\limits_{n \to \infty } {a_n}\exists \\ 令\mathop {\lim }\limits_{n \to \infty } {a_n} = A\\ {a_{n + 1}} = \sqrt {2 + {a_n}} \Rightarrow A = \sqrt {2 + A} \\ \Rightarrow {A^2} - A - 2 = 0\\ {A_1} = - 1(舍去);{A_2} = 2\\ \end{array}\]

类型三 不定型求极限（重点）

  • 共七种类型（$\frac{0}{0},{1^\infty },\frac{\infty }{\infty },0*\infty ,\infty - \infty ,{\infty ^0},{0^0}$）
  • $\frac{0}{0}型（重点）$
    思路：
\[\begin{array}{*{20}{l}} {1.u_{(x)}^{{v_{(x)}}} \Rightarrow {e^{v(x)\ln (x)}}}\\ {2.\ln () \Rightarrow \ln (1 + \Delta )\;\Delta \to 0}\\ {3.() - 1 \Rightarrow \{ _{{{(1 + \Delta )}^a} - 1\;a\Delta }^{{e^\Delta } - 1\;\Delta }}\\ {4.x - \ln (1 + x) \Rightarrow 二阶无穷小}\\ {5.x,\sin x,\tan x,\arcsin x,\arctan x任意两个相减为三阶无穷小} \end{array}\]

误区
$\frac{0}{0}$ 型中使用等价无穷小，在乘法中可以随便使用，在加法中，需要分子分母次数相同

  • ${1^\infty }型（重点）$
    思路：
$1.{(1 + \Delta )^{\frac{1}{\Delta }}}$

2.恒等变形

  • $\infty - \infty型$
    思路：

1.有分母通分。
2.没有分母（根式），分子有理化
3.转化

例1： \[\begin{array}{l} \mathop {\lim }\limits_{x \to 0} (\frac{1}{{{x^2}}} - \frac{1}{{{{\tan }^2}x}})\\ = \mathop {\lim }\limits_{x \to 0} (\frac{{{{\tan }^2}x - {x^2}}}{{{x^4}}})\\ = \mathop {\lim }\limits_{x \to 0} \frac{{\tan x{\rm{ + x}}}}{x} + \mathop {\lim }\limits_{x \to 0} \frac{{\tan x - {\rm{x}}}}{{{x^3}}}\\ = 2\mathop {\lim }\limits_{x \to 0} \frac{{{{\sec }^2}x - 1}}{{3{x^2}}}\\ = \frac{2}{3}\mathop {\lim }\limits_{x \to 0} \frac{{{{\tan }^2}x}}{{{x^2}}}\\ = \frac{2}{3} \end{array}\]
例2： \[\begin{array}{l} \mathop {\lim }\limits_{x \to \infty } (\sqrt {{x^2} - 4x + 8} - x)\\ = \mathop {\lim }\limits_{x \to \infty } \frac{{ - 4x + 8}}{{\sqrt {{x^2} - 4x + 8} + x}}\\ = \mathop {\lim }\limits_{x \to \infty } \frac{{ - 4 + \frac{8}{x}}}{{\sqrt {1 - \frac{4}{x} + \frac{8}{{{x^2}}}} + 1}} \end{array}\]

  • $\frac{\infty }{\infty }型$
    一.思路：

1.$\mathop {\lim }\limits_{x \to \infty } \frac{{{b_m}{x^m}{\rm{ + }}......{\rm{ + }}{{\rm{b}}_0}}}{{{a_n}{x^n} + ...... + {a_0}}}$

当m<n时$\to$原式=0

当m=n时$\to$原式=$\frac{{{b_m}}}{{{a_n}}}$

当m>n时$\to$原式=$\infty$
2.罗氏法则（洛必达法则）

一.小知识：

$1.\mathop {\lim }\limits_{x \to \infty } \frac{{{x^3}}}{{{e^x}}} = \mathop {\lim }\limits_{x \to \infty } \frac{{3{x^2}}}{{{e^x}}} = \mathop {\lim }\limits_{x \to \infty } \frac{6}{{{e^x}}} = 0$

指数函数走向无穷的速度要比幂函数快

$2.\mathop {\lim }\limits_{x \to \infty } \frac{{\ln x}}{{\sqrt x }} = \mathop {\lim }\limits_{x \to \infty } \frac{{\frac{1}{x}}}{{\frac{1}{{2\sqrt x }}}} = \mathop {\lim }\limits_{x \to \infty } \frac{2}{{\sqrt x }} = 0$

幂函数走向无穷的速度要比对数函数快
conclusion:走向无穷的速度：对数函数<幂函数<指数函数

  • $0*\infty型$
    思路：

转化

$1.\frac{0}{{\frac{1}{\infty }}}$即$\frac{0}{0}型$
$2.\frac{\infty }{{\frac{1}{0}}}$即$\frac{\infty }{\infty }型$

例： \[\begin{array}{l} \mathop {\lim }\limits_{x \to \infty } {(2x + 1)^2}\sin \frac{1}{{{x^2}}}\\ = \mathop {\lim }\limits_{x \to \infty } \frac{{{x^2}}}{{{{(2x + 1)}^2}}}*\frac{{\sin \frac{1}{{{x^2}}}}}{{\frac{1}{{{x^2}}}}}\\ = \mathop {\lim }\limits_{x \to \infty } \frac{{{x^2}}}{{4{x^2} + 4x + 1}}\\ = \frac{1}{4} \end{array}\]

类型四 左右极限

${a^{\frac{\Delta }{{x - b}}}},{a^{\frac{\Delta }{{b - x}}}}$当$x \to b$时一定分左右讨论

例： \[\begin{array}{l} f(x) = {e^{\frac{x}{{x - 2}}}},\mathop {\lim }\limits_{x \to 2} f(x)\\ f(2 - 0) = 0,f(2 + 0) = + \infty \\ \therefore \mathop {\lim }\limits_{x \to 2} f(x)不存在 \end{array}\]

间断

思路：

1.找间断点(没有定义，无法取到的点)
2.指数是这样的形式：${a^{\frac{\Delta }{{x - b}}}}$，分左右极限讨论。
-介值定理（重点）
思路：
当$f(x) \in [a,b]$有$\zeta \in [a,b]$和函数值之和时$\Rightarrow$介值定理
例： \[\begin{array}{l} f(x) \in [a,b],p > 0,q > 0,p + q = 1;\\ 证明：\exists \zeta \in [a,b],f(\zeta ) = pf(a) + qf(b)\\ 解：f(x) \in [a,b] \Rightarrow \exists m,M\\ m = pm + qm \le pf(a) + qf(b) \le pM + qM \le M\\ \therefore \exists \zeta \in [a,b],f(\zeta ) = pf(a) + qf(b)\\ \end{array}\]
