导数与微分

发表于 2020-03-31 更新于 2020-04-02 分类于 考研 阅读次数： Valine：
汤神的导数与微分部分视频已经看完，总结出如下重要的内容

重要知识点

  • 1.可导必连续，连续不一定可导。
  • 2.可导可以等价于可微。$dy = f'(x)dx$
  • 3.看到$f(x)连续{\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} \mathop {\lim }\limits_{x \to a} \frac{{f(x) - b}}{{x - a}} = A$必要想到：
    $f(a) = b,f'(a) = A$
  • 4.等待补充

类型一 显函数求导

  • 没有难度，直接求导。

类型二 复合函数求导

  • 利用公式，四则求导法则，一层一层的求导。

类型三 隐函数求导（重点）

\[\begin{array}{l} {\rm{y}} = f(x) \to显函数 \\ F(x,y) = 0 \to 隐函数 \end{array}\]

例：

\[\begin{array}{l} {e^{xy}} = {x^2} + {y^2} + 1{\kern 1pt} {\kern 1pt} 求{\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} {\kern 1pt} \frac{{dy}}{{dx}}\\ \Rightarrow {e^{xy}}(y + xy') = 2x + 2yy' \end{array}\]
